# TypeScript Quiz — ข้อ 5 (1 คะแนน)
## 🎯 คำสั่ง

สร้าง class ตาม UML diagram ที่ออกแบบไว้ โดยต้องทำให้ไฟล์ `index.ts` ที่กำหนดให้ สามารถทำงานได้ถูกต้อง และได้ผลลัพธ์ตามที่คาดหวัง

ห้ามแก้ไขไฟล์ `index.ts`

## 📌 หมายเหตุสำคัญ

**ในไฟล์ `index.ts` มี comment (`//`) ที่แสดงตัวอย่างผลลัพธ์ที่คาดหวัง**

ให้นักเรียนอ่านผลลัพธ์จาก comment เหล่านั้น แล้วเขียน method logic ใน class ต่างๆ ให้สอดคล้องกับผลลัพธ์ที่แสดงไว้

**ตัวอย่าง:**
```typescript
circle1.draw()
// Drawing a red circle with radius 5
```

จาก comment ด้านบน แสดงว่าผลลัพธ์ที่คาดหวังคือ `"Drawing a red circle with radius 5"` ดังนั้น method `draw()` ใน class `Circle` ต้องแสดงผลลัพธ์ในรูปแบบนี้

```
📁 โครงสร้างไฟล์ที่ต้องมี
p05/
 ├── shape.ts        ← นักเรียนต้องสร้าง
 ├── circle.ts       ← นักเรียนต้องสร้าง
 ├── rectangle.ts    ← นักเรียนต้องสร้าง
 ├── drawable.ts     ← นักเรียนต้องสร้าง
 ├── index.ts        ← ไฟล์ที่กำหนดให้
 └── uml.png         ← UML diagram
```
